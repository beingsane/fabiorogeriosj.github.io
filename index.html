Sobre o treinamento: AngularJS em Ação

O treinamento em AngularJS tem como objetivo aprender e praticar, de forma direta, as principais funcionalidades deste framework fantástico!
Para realizar o treinamento é necessário conhecimento básico na linguagem JavaScript.

Conteúdo:

  • Introdução ao AngularJS Explicação dos conceitos de AngularJS, para que ele foi criado, o que é SPA, etc.
  • Directives Directive é uma maneira de criar novas funcionalidades em HTML.
  • Data Binding A magia do AngularJS esta em seu Data Binding, onde os elementos HTML são atualizados automático quando um modelo é alterado.
  • Filters Uma forma simples de filtrar valores em objetos.
  • Views Páginas HTML que é manipulada pelos binding do AngularJS.
  • Controllers Um controller é um objeto JavaScript contendo atributos e ações.
  • Scope Scope é o responsável por controlador ligar os objetos do controller com as views.
  • Modules AngularJS suporta módulos externos do seu Core, semelhante a plugin.
  • Routes Controlar as rotas de diferentes Views e Controllers.
  • Factories e Services Encapsulamento de funcionalidades que podem ser invocados entre diferentes controllers.
